<h1>Creating First Subscription and First Resource Group </h1>

- Microsoft Azure

<h2>Operating Systems Used </h2>

- Windows 10 (21H2)
  
<h2>High-Level Steps</h2>

- Azure.com
- Registration
- Azure Portal
- Creation of Resource Group
